# Domain Değerleme ve Valuation Rehberi

## 1. Değerleme Algoritmaları ve Araçları (2026)

### Ana Faktörler
- Uzunluk, keyword kalitesi, TLD (.com öncelik)
- Comparable sales + ML pattern recognition
- Backlink/SEO metrikleri (expired için kritik)
- Ticari potansiyel (niş uyumu: Alüminyum, Marine)

### Önerilen Araçlar
- **GoDaddy Domain Appraisal**: ML + satış verisi
- **EstiBot**: 100+ attribute, tutarlı tahmin
- **NameWorth / HumbleWorth**: AI tabanlı

**Kullanım**: Bir domain için GoDaddy + EstiBot ikilisini kullan, farkı analiz et.

## 2. Domain Değerleme Checklist

- [ ] Uzunluk < 15 karakter mi?
- [ ] .com veya güçlü TLD mi?
- [ ] Dictionary / brandable / pronounceable mı?
- [ ] Keyword arama hacmi yüksek mi? (Alüminyum/Marine nişi)
- [ ] Backlink kalitesi (BL > 100, temiz history)
- [ ] Archive yaşı (ABY) eski mi?
- [ ] Spam/penalty yok mu?
- [ ] Comparable sales var mı? (EstiBot/GoDaddy)
- [ ] İş potansiyeli (kendi markan için mi, flipping mi?)
- [ ] Trademark riski kontrol edildi mi?

## 3. Alüminyum Nişi İçin Değerleme Kriterleri

**Yüksek Değer Potansiyeli**:
- "Alu", "Aluminum", "Billet", "Ingot", "Scrap", "Alloy", "MetalTrade" içeren domainler
- Kısa, markable (.com)
- LME, trading, global ticaret çağrışımı

**Örnek Değerlendirme**:
- AluTradeHub.com → Yüksek (keyword + brandable)
- BilletDirect.com → Orta-Yüksek

**Düşük Değer**:
- Çok uzun, alakasız kelimeler, spam history
